# @area44/biome-config

## Installation

You can install `@area44/biome-config` follows:  

```sh
pnpm i -D @area44/biome-config
```

## Configuration

To apply the configuration, update your `biome.json` file as follows:

```json
{
  "extends": ["@area44/biome-config"]
}
```
